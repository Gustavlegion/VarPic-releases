# VarPic

**Image Similarity Search Tool** — find visually similar images using a reference image.

## Download

[![Download](https://img.shields.io/github/v/release/Gustavlegion/VarPic-releases?label=Download&style=for-the-badge)](https://github.com/Gustavlegion/VarPic-releases/releases/latest)

## Requirements

- Windows 10 / 11 (64-bit)
- ~250 MB free disk space

 (Linux)                                                                                                          
  - Ubuntu 20.04 або новіше (64-bit)
  - Debian 11 (Bullseye) або новіше

## Installation

1. Download `VarPic_Setup_vX.X.X.exe/.deb` from [Releases](https://github.com/Gustavlegion/VarPic-releases/releases/latest)
2. Run the installer
3. Follow the on-screen instructions
